# keyboard_manager_example

Demonstrates how to use the keyboard_manager plugin.

## Installation



```dart
pip install foobar
```

## Usage

```python
import foobar

foobar.pluralize('word') # returns 'words'
foobar.pluralize('goose') # returns 'geese'
foobar.singularize('phenomena') # returns 'phenomenon'
```
